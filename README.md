# Messenger App with OTP Authentication

A secure messaging application with OTP-based password reset and social authentication.

## Features

- User registration and authentication
- Social login (Google, Facebook)
- OTP-based password reset via Email/SMS
- OTP-based password change
- Account verification with OTP
- Real-time messaging
- Group chats
- Friend system
- File sharing
- Emoji support
